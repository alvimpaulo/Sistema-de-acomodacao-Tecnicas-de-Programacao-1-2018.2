Detailed Description

Domínio armazena uma data.

Domínio armazena uma data conforme o formato "DD/MMM/AAAA", onde DD é um número de 01 a 31, MMM é a inicial dos meses do ano (jan, fev, mar, abr, mai, jun, jul, ago, set, out, nov ou dez) e AAAA é um número de 2000 a 2099.
