Detailed Description

Domínio que armazena uma data de validade.

Domínio que armazena uma data de validade Essa classe armazena uma data de validade do cartão de crédito no formato “MM/AA”, onde MM é um número entre 01 e 12 composto por 2 algarismos e AA é um número entre 00 e 99 composto número composto, também, por 2 algarismos.
